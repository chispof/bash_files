# firewallSearch
Scripts for Palo Alto Firewall management

chmod +x firewallSearch.sh

## Update information

./firewallSearch.sh -u

## Search by object

./firewallSearch.sh -o IP|Object_name

## Search by source

./firewallSearch.sh -s IP|Object_name

## Search by rule name

./firewallSearch.sh -n Rule_name
